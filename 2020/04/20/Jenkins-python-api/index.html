前言

总结

建议使用Python Jenkins模块，相对JenkinsApi而言，模块封装的更好。接口调用更方便，更容易上手,模块内容不是很多，可以自己研读源码进行二次开发，方便与其他系统集成，更好的落地DevOps方案的实施;
